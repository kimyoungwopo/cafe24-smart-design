⚡ 설치 방법

Claude Code
스킬 디렉토리에 클론하면 자동 트리거
git clone https://github.com/kimyoungwopo/cafe24-smart-design.git \ ~/.claude/skills/cafe24-smart-design
Cursor
.cursor/rules/ 에 레퍼런스 복사
mkdir -p .cursor/rules cp references/*.md .cursor/rules/ cp SKILL.md .cursor/rules/cafe24.md
GitHub Copilot
.github/copilot-instructions.md 에 병합
mkdir -p .github cat SKILL.md references/*.md \ > .github/copilot-instructions.md
Windsurf
.windsurfrules 파일로 병합
cat SKILL.md references/*.md \ > .windsurfrules
GPT / Codex
Custom Instructions 또는 Knowledge 업로드
ChatGPT → Settings → Custom Instructions GPT Builder → Knowledge 파일 업로드 Codex → AGENTS.md에 포함
Gemini
Gems 또는 System Instructions에 추가
Gems → Instructions에 붙여넣기 AI Studio → System Instructions Gemini CLI → GEMINI.md에 포함

모듈용도주요 변수

코드 예시

      <a href="{$link_product_detail}{$param}">
        <img src="{$image_medium}" alt="{$product_name}">
        <p>{$product_name|cut:25,...}</p>
        <span>{$product_price|numberformat}원</span>
      </a>
    </li>
  </ul>
</div>
